Для корректной работы приложения необходимо в исполняемую директорию добавить файл loginID.txt
В нём на первой строке должна содержаться почта под которой осуществляется вход.
На второй строке должен содержаться хэш пароля (можно взять из консоли разработчика при отправке запроса login).

Пример содержимого:
username@mail.aaa
askdfjasdfj=243n!-==
